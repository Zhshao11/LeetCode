# 二分算法
- 问题：查找有序数组中元素小于k的数的数组长度
- 使用二分查找
  - 左闭右闭区间：R初始为数组长度-1，更新时L更新为M+1，R更新为M-1，最后L=R时还需要遍历一次，返回L
    - 不变量：L的左边全是比k小的数，R右边全是大于等于k的数
  - 左闭右开区间：R初始化为数组长度，更新时L更新为M+1,R更新为M，L=R时退出循环，最后返回L或R都行
  - 左开右开区间：L初始化为-1，R初始化为数组长度，最后L+1=R时退出循环，返回值为R
- 怎么判断是哪种二分？
  - 看while的条件：
    - left <= right:right最后跑到了left左边，返回值是left，是左闭右闭区间
    - left < right:right等于left时退出循环，最后left=right，返回left和right都行，是左闭右开区间
    - left+1 < right:left在right左边退出循环，最后是left+1=right,返回right，是左开右开区间
- 如何防止加法溢出？
  - M = (L+R)/2改为M = L+(R-L)/2

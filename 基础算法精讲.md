# 相向双指针
## 两数之和、三数之和
- 先进行**排序**，然后根据有序性使用相向双指针，一个指针从头开始，一个指针从尾开始，并加以优化：各种特殊情况。
## 盛水最多的容器
- 相向双指针，每次只移动最短边
## 接雨水
- 前后缀分解：每次雨水的量等于min(前板max、后板max)，得到前板的最大值和后板的最大值
- 相向双指针：依次遍历前后指针，前板和后板哪边小就移动哪边的指针
# 滑动窗口（窗口大小固定）、双指针（窗口大小不固定）
# 同向双指针
- 移动右端点，收缩左端点
- 时间复杂度是O(N)，左端点和右端点都是从1到n，在双重循环里没有重置左端点
- 什么时候才能使用双指针？
  - 循环是具有单调性的：从满足要求变成不满足要求
## 643. 子数组最大平均数 I
- 返回值：使用int/int得到的最后还是int，如果 返回值类型是double，会损失精度，需要使用1.0*int/int得到double类型
## 1343. 大小为 K 且平均值大于等于阈值的子数组数目

- ```
  使用1.0*sum/k >= threshold比使用sum >= k*threshold慢2ms
  ```
